DirLens

A free, native image viewer for Mac. Open one photo, browse the whole folder, zoom, rotate — the way image viewing used to just work.

Download for Mac

macOS 14 Sonoma or later · Apple Silicon & Intel · Free · View source on GitHub

🗂️

Folder aware

Open any image and browse every other image in the same folder automatically.

🔍

Zoom & pan

Pinch to zoom, drag to pan, double-click to reset — smooth and instant.

↻

Rotate

Rotate left or right — saves losslessly to the actual file, not just the view.

🎞️

Thumbnail filmstrip

Hideable bottom drawer with thumbnails of the whole folder. Click to jump.

⌨️

Keyboard first

Arrow keys to move, space to toggle the filmstrip, no mouse required.

✋

Drag & drop

Drop an image onto the window to open it. No import, no library.

🗑️

Move to Trash

Delete the current image straight from the viewer — safely, via the Trash, with a confirmation first.

Install

  1. Download the .zip above and unzip it.
  2. Drag DirLens.app into your Applications folder.
  3. Right-click (Control-click) DirLens.app and choose Open, then confirm Open in the dialog.
About that Gatekeeper warning DirLens is free and distributed without an Apple Developer certificate, so macOS will say it "cannot be opened because Apple cannot check it for malicious software" on first launch. The right-click → Open step above bypasses that — you only need to do it once. Alternatively, run xattr -dr com.apple.quarantine /Applications/DirLens.app in Terminal.

On newer macOS versions, right-click → Open may still show a dialog with only "Move to Trash" and "Done" (no "Open Anyway"). If so, go to System Settings → Privacy & Security, scroll down to the message about DirLens, and click Open Anyway — then try opening it again.

Folder-access permissions (Desktop, Documents, Downloads, etc.) are a separate, one-time prompt from macOS — once you grant one, it stays granted across DirLens updates.
